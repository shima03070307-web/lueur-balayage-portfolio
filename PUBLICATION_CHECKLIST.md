# 公開前チェック

## GitHub Pages設定

- [ ] リポジトリ直下に `index.html` がある
- [ ] `assets/` にWebP画像とアイコン画像がある
- [ ] `Settings` -> `Pages` で公開元が `main` / `/root` になっている
- [ ] 公開URLで `index.html` と `case-study.html` が開く

## 表示確認

- [ ] スマホ幅 360px で横スクロールが出ない
- [ ] スマホ幅 390px で見出し、バッジ、ボタン、料金タグが不自然に改行されない
- [ ] PC幅 1366px でファーストビューのCTAが見える
- [ ] 画像がすべて表示される
- [ ] メニュー、FAQ、ページ内リンクが動く

## 公開URL確定後

- [ ] 制作事例ページに canonical を追加する
- [ ] `og:url` を追加する
- [ ] 必要なら `sitemap.xml` を公開URLで作る
- [ ] 実際の問い合わせ先が決まったらCTAを差し替える
- [ ] 架空サイト表記が残っていることを確認する
